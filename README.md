# You Became The Money

AI command center for Daniel Garcia's commercial law practice.

## Quick Start for Developers

**⚠️ READ FIRST:** [CLAUDE.md](CLAUDE.md) - Workspace routing and structure

### Project Structure

```
├── CLAUDE.md          ← START HERE (routing rules)
├── CONTEXT.md         ← Project requirements
├── BUILD.md           ← Task checklist  
├── SKILL.md           ← Tech stack & skills
├── prototype/         ← Phase 1: Dashboard MVP
├── integration/       ← Phase 2: Hermes + API
├── optimization/      ← Phase 3: Production
├── index.html         ← Marketing website
└── about.html         ← About page
```

### Run Prototype Dashboard

```bash
cd prototype
npm install
npm run dev
```

Open http://localhost:3000

### Tech Stack

- React + TypeScript + Tailwind CSS
- Node.js API (Express)
- PostgreSQL
- Hermes Agent (AI orchestrator)
- FastMCP (custom skills)
