## Troubleshooting

### Dashboard not loading?
1. Refresh page (Cmd/Ctrl + R)
2. Check internet connection
3. Clear browser cache
4. Try different browser

### No leads showing?
1. Ensure Retell agent is deployed
2. Check that leads are being created
3. Verify database connection
4. Review API logs

### Voice commands not working?
1. Ensure browser allows microphone access
2. Check system volume
3. Try different browser (Chrome works best)
4. Verify sound output enabled

### Social metrics not updating?
1. Check Instagram/Facebook connections
2. Verify API credentials
3. Wait up to 1 hour for sync
4. Manually refresh page
